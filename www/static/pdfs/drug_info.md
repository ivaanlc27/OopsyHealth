# Drug information index

This file explains how the drug information files are organised in this lab and lists the drug summaries currently available.

## How to find drug information

Each drug summary is stored as a separate file. To find the information for a drug, use the following filename pattern: drug_name_info.md

## Available drug summaries in this lab

- **aspirin** — `aspirin_info.md`
  Short: analgesic, antipyretic, anti-platelet agent used in low doses for cardiovascular prophylaxis.

- **amoxicillin** — `amoxicillin_info.md` 
  Short: aminopenicillin antibiotic used for a variety of common bacterial infections.
